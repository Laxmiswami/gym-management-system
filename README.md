# Gym Management System

## Description
A full-stack web application for managing gym operations such as trainers, diet plans, and workout plans.

## Tech Stack
- Frontend: HTML, CSS, Bootstrap
- Backend: Java, JDBC
- Database: MySQL

## Features
- Secure Login & Registration
- Trainer Management
- Diet Plan Assignment
- Workout Plan Management
- Role-based Access (Admin/Trainer)
